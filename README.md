# Formatar Processos CNJ

Automação desenvolvida em Python para padronizar números de processos no formato CNJ utilizando pandas.

## Funcionalidades

- Completa números de processo com zeros à esquerda até 20 caracteres.
- Formata o número para o padrão CNJ.
- Lê dados de planilhas Excel.
- Exporta o resultado para um novo arquivo Excel.
